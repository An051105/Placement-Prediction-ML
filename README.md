# 📊 Placement Prediction using Machine Learning

## 🚀 Project Overview

This project predicts whether a student will be placed based on academic performance and test scores.
It is a **classification problem** solved using Machine Learning.

---

## 🎯 Objective

To analyze student data and build a model that can predict placement outcomes based on key academic features.

---

## 📁 Dataset

* Source: Campus Placement Dataset (Kaggle)
* Total records: ~215 students
* Features include:

  * SSC Percentage
  * HSC Percentage
  * Degree Percentage
  * Employability Test Score
  * MBA Percentage

---

## ⚙️ Technologies Used

* Python
* Pandas
* NumPy
* Scikit-learn
* Matplotlib
* Seaborn

---

## 🧠 Machine Learning Approach

* Problem Type: **Binary Classification**
* Model Used: **Logistic Regression**
* Steps:

  * Data Cleaning (handling missing values)
  * Feature Selection
  * Train-Test Split
  * Model Training
  * Evaluation using Accuracy

---

## 📈 Results

* Achieved **~88% accuracy** on test data

---

## 📊 Key Insights

* Academic performance (SSC, HSC, Degree) has a strong impact on placement
* MBA percentage and test scores showed relatively lower influence
* Higher academic scores increase probability of placement

---

## 📌 Visualizations

* Correlation Heatmap
* Boxplot for SSC vs Placement
* Scatter Plot for feature relationships

---
